## Commands

To run this repo, you may need to run these commands.

- Install `node_modules` folder:
```
npm install
```
- Install any missing libraries:
```
npm install --save react-router-dom
npm install bootstrap jquery
npm install axios
npm install json-server
```
- Run the database: 
```
npm run server
```
- Run the program: 
```
npm start
```
